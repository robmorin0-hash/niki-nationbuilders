Solar System Installation and Commissioning: A Practical Guide

Grounded in primary-source research. Source: solar-pv-systems-design-installation-depth.

Pre-Installation Planning: Laying the Foundation

Before a single panel is mounted, meticulous planning ensures a safe, efficient, and compliant installation. Begin with a site survey to analyze shading patterns, accounting for 30+ year tree growth projections. Verify your roof’s load capacity, requiring a structural engineer for systems exceeding 10 kW rooftop. Confirm access to your electrical service panel, noting conduit routing and distance from the array. Position equipment like the inverter, disconnect, and breaker box strategically to minimize cable runs. Finally, secure jurisdiction-specific permits, which typically take 2–4 weeks to approve.

Documentation is critical: prepare a system one-line diagram detailing AC/DC circuits and equipment ratings. Include structural design for roof load and racking attachment, along with electrical calculations for wire sizing, voltage drop, fault current, and ground electrode design. Submit module/inverter/charger specifications with datasheets and certifications. Estimate system capacity and expected yield using tools like the NSRDB or PVGIS databases.

Installation Workflow: Step-by-Step Execution

Day 1–2: Roof or Ground Preparation Mount racking systems, ensuring roof flashing is sealed for waterproofing and ground footings are set for stability. Verify structural attachment torque matches manufacturer specs (typically 50–150 in-lbs). Install conduits for AC/DC runs from the array to the inverter location, avoiding sharp bends or obstacles.

Day 3: Module Installation Install rails, clamps, and modules, ensuring alignment and spacing. Assemble cables with junction boxes and use parallel combiners for multiple strings. Measure Voc on each string to confirm it matches the expected value (equal to Voc_rated × number of modules at 25°C, adjusted for temperature deviations).

Day 4: DC System Commissioning Test string currents to ensure they are equal and within 10% mismatch (excess indicates shading or defective modules). Perform an insulation resistance test using a 500 VDC megohmmeter, requiring >1 GΩ DC ground resistance for each string. Verify open-circuit voltage matches calculated values adjusted for temperature.

Day 5: Inverter Commissioning Connect DC wiring with polarity verified and install a high-voltage hazard warning label. For AC connections, install a utility/grid breaker and a transfer switch if a backup system is present. Program inverter firmware with voltage/frequency settings, islanding parameters for grid-tied systems, and charge/discharge setpoints for battery integration.

System Activation and Optimization

Day 6: Final Checks and Optimization Power up the system for no-load testing, verifying LED/display indicators. Gradually apply loads to measure current/voltage and confirm efficiency. Perform IV curve tracing (optional) using high-precision tools to validate module output against STC values. Use thermal imaging (optional) to detect hot cells, indicating partial shading or defects.

Inspection and Approval: Finalizing the Process

After installation, schedule an Authority Having Jurisdiction (AHJ) inspection, typically within a week of completion. For grid-tied systems, a utility inspection may be required, though it’s often a review of the one-line diagram by an engineer. Obtain the final permission to operate (PTO) document before activating net metering.

Gear that Helps

  • A solar racking system with sealed roof flashing
  • A torque wrench for verifying structural attachment torque (50–150 in-lbs)
  • A combiner box rated for the array's string voltage (e.g., 1000V DC)
  • A digital multimeter (for voltage/current checks)
  • A 500 VDC megohmmeter (insulation resistance testing)
  • A thermal imaging camera (for hot-spot detection)
