Sizing Solar Arrays and Designing String Configurations for Off-Grid Systems

Grounded in primary-source research. Source: solar-pv-systems-design-installation-depth.

Example residential home (Alberta, electric heating):

  • Summer: 20 kWh/day (AC cooling dominant)
  • Winter: 40 kWh/day (space heating, water heating)
  • Spring/Fall: 30 kWh/day
  • Annual average: 30 kWh/day

To size a solar array, start by calculating the daily load. For off-grid systems, prioritize winter conditions. If grid-tied, size for summer with battery backup; for off-grid, size for winter minimum.

---

Load Assessment and Array Sizing

Step 1: Determine daily load Use historical data to calculate average daily energy consumption. For Alberta, the annual average is 30 kWh/day.

Step 2: Apply the array sizing formula Array capacity (kW) = Daily load (kWh) / [PSH × System efficiency] Where:

  • Daily load: 30 kWh/day
  • PSH: 4.0 peak sun hours/day (Alberta annual average)
  • System efficiency: 0.75 (includes inverter 95%, DC wiring 97%, controller 96%, soiling 98%, temperature derating 82% of STC performance at NOCT)

Calculation: 30 kWh / (4.0 × 0.75) = 10 kW array nameplate capacity

Account for seasonal variation:

  • Winter requires 17.8 kW (40 kWh/day, 3.0 PSH, 0.75 efficiency).
  • Grid-tied systems: 10 kW is adequate with battery backup for summer.
  • Off-grid systems: Size for winter minimum (18 kW) with a 30+ kWh battery bank.

---

String Configuration and Electrical Design

Series string design (voltage) Use modules with 37V Voc and -0.32%/°C temperature coefficient.

Cold temperature adjustment (NEC 690.7):

  • At -40°C ambient, module Voc increases by 20.8% (37V × 1.208 = 44.7V).
  • A 25-module series string would reach 1,118V (25 × 44.7V), exceeding MPPT input limits.

MPPT input voltage range:

  • Configure as two parallel 12-module strings (12 × 44.7V = 536V) or use a 600V MPPT with a 13-module string (13 × 44.7V = 581V).

Parallel string design (current) Each module has 11A Isc. Two parallel strings yield 22A total current into the MPPT input.

Wire sizing (DC side) For a 100 ft run:

  • Conductor size = 1.25 × Isc / voltage drop (2% of 600V = 12V).
  • Required resistance: 12V / 22A = 0.545 Ω.
  • #2/0 AWG copper provides about 0.198 Ω over the run (~1.3% drop), meeting the 2% voltage-drop requirement.

Actual specification: #2/0 AWG copper, PV-rated UV-resistant cable, 600V rated, conduit raceway.
